# Week 01 Brain 60 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expand the `brain` question bank in `apps/web/public/question-packs/week-01.json` from 12 questions to 60 without changing other weeks or other subjects.

**Architecture:** Keep the existing schema and quiz flow unchanged. Update the focused week-01 brain-bank test first, then append 48 additional `brain` questions using the existing `choice` and `input` structures, and finally verify both focused data tests and broader schema/build checks.

**Tech Stack:** JSON question packs, TypeScript, Vitest, Vite, Zod

### Task 1: Raise the week-01 brain-bank contract to 60 questions

**Files:**
- Modify: `apps/web/src/data/__tests__/week01BrainBank.test.ts`
- Test: `apps/web/src/data/__tests__/week01BrainBank.test.ts`

- [ ] **Step 1: Write the failing test**

```ts
import { describe, expect, it } from 'vitest';

import pack from '../../../public/question-packs/week-01.json';

type Q = {
  subject: string;
  topic: string;
  type: string;
  options?: string[];
  answer: string;
};

const brainQuestions = (pack.questions as Q[]).filter((q) => q.subject === 'brain');

describe('week-01 脑筋急转弯 bank', () => {
  it('contains exactly 60 brain questions', () => {
    expect(brainQuestions).toHaveLength(60);
  });

  it('covers at least 6 distinct topics', () => {
    const topics = new Set(brainQuestions.map((q) => q.topic));
    expect(topics.size).toBeGreaterThanOrEqual(6);
  });

  it('uses only supported brain question types', () => {
    brainQuestions.forEach((q) => expect(['choice', 'input']).toContain(q.type));
  });

  it('every choice/image-choice answer appears in options', () => {
    brainQuestions
      .filter((q) => q.type === 'choice' || q.type === 'image-choice')
      .forEach((q) => expect(q.options).toContain(q.answer));
  });
});
```

- [ ] **Step 2: Run test to verify it fails**

Run: `pnpm --filter @ultraman/web exec vitest run src/data/__tests__/week01BrainBank.test.ts`
Expected: FAIL because the current bank contains 12 brain questions instead of 60

```bash

git commit -m "test: raise week-01 brain bank target to 60"
```

### Task 2: Append 48 new brain questions to week-01

**Files:**
- Modify: `apps/web/public/question-packs/week-01.json`
- Test: `apps/web/src/data/__tests__/week01BrainBank.test.ts`

- [ ] **Step 1: Add the new questions**

Append entries `b-w01-013` through `b-w01-060` under `questions` in `apps/web/public/question-packs/week-01.json` using this shape:

```json
{
  "id": "b-w01-013",
  "subject": "brain",
  "difficulty": 1,
  "topic": "文字拆解",
  "type": "choice",
  "stem": "什么字一写就错？",
  "options": ["错", "对", "好", "写"],
  "answer": "错"
}
```

Rules for the full set:

- Keep existing `b-w01-001` to `b-w01-012` unchanged
- New IDs must be continuous through `b-w01-060`
- Use only `choice` and `input`
- Cover at least these topics: `谐音`, `逻辑`, `数字`, `生活反差`, `文字拆解`, `观察判断`
- Prefer `difficulty` 1 or 2 only
- Use `input` only when the answer is a single stable string such as `1`, `2`, `5`, `一`, `火`
