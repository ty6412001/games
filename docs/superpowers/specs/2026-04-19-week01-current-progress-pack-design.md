# week-01 当前教材进度包设计

**目标**

将现有 `apps/web/public/question-packs/week-01.json` 从“通用综合卷”重写为“当前教材进度包”，使其与当前教学进度一致：

- 语文：苏教版一年级下册学完 `练习4`
- 数学：苏教版一年级下册对齐前 `3` 个板块
- 英语：译林版一年级下册 `1B Unit 1-4`

**范围**

- 重写 `apps/web/public/question-packs/week-01.json`
- 更新 `apps/web/public/question-packs/index.json` 中 `week-01` 的标题
- 不修改 `week-02` 到 `week-10`
- 不修改前端题型、判题逻辑、题包加载逻辑

**教材锚点**

- 语文锚点：苏教版一年级下册到 `课文13《好学的爸爸》` 与 `练习4`
- 数学锚点：苏教版一年级下册前 `3` 个板块
  - `20以内的退位减法`
  - `认识图形（二）`
  - `认识100以内的数`
- 英语锚点：译林版一年级下册 `1B Unit 1-4`
  - `Unit 1 Let's count`
  - `Unit 2 This is my pencil`
  - `Unit 3 I like carrots`
  - `Unit 4 Spring`

说明：

- 语文进度是用户明确给定
- 数学与英语进度按同阶段教材目录匹配，是对当前时间点的合理对齐

**题量与分配**

- 每周总题量保持 `95`
- 数学 `40` 题
- 语文 `40` 题
- 英语 `15` 题

数学内部分配：

- `20以内退位减法`：`16` 题
- `认识图形（二）`：`12` 题
- `认识100以内的数`：`12` 题

语文范围：

- `识字1-4`
- `课文1-13`
- `练习1-4` 的基础能力

语文内部分配：

- `识字1-4`：`10` 题
- `课文1-4`：`6` 题
- `课文5-8`：`8` 题
- `课文9-13`：`10` 题
- `练习1-4`：`6` 题

英语范围：

- 数字 `1-10`
- 文具词汇
- 食物/喜好
- 春天相关基础表达

**题型策略**

继续只使用现有四种题型：

- `choice`
- `input`
- `image-choice`
- `ordering`

同时收紧出题规则：

- 不使用多解 `input` 题
- 不使用带重复项的 `ordering` 题
- 题型以 `choice` 为主，数学与语文都优先扩充不同问法的选择题
- 不重复完全相同的题干，也不做仅换数字的低质量堆题
- `choice` / `image-choice` 的 `answer` 必须出现在 `options`
- `ordering.answer` 必须等于 `correctOrder.join(',')`
- 英语输入题仅使用单词或固定短语
- 语文题尽量锚定课文人物、词语、句子理解，不再使用脱离课文的通用拼音专题

**命名与标题**

- 保持 `week-01.json` 文件名不变
- 题目 id 继续沿用 `<学科首字母>-w01-<序号三位>`
- `week-01` 标题改为带教材进度说明的专题包标题
  - 建议标题：`第1周 · 当前教材进度包 / 语文至练习4 / 数学前三单元 / 英语1B Unit1-4`

**验证**

- 用 `QuestionPackSchema` 校验重写后的 `week-01.json`
- 用 `QuestionPackIndexSchema` 校验 `index.json`
- 运行 `pnpm --filter @ultraman/shared run build`
- 运行 `pnpm --filter @ultraman/web run build`

**非目标**

- 不追求逐课逐页一一对应
- 不改造判题系统以支持同义答案
- 不补充新的图片素材
- 不重排 `week-02` 以后题包
